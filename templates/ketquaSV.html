Logic Error Case #04: Race Condition in Async Loop

main.js Line 142:24

async function processData(items) {

items.forEach(async (item) => {

// Error: forEach doesn't wait for await

await saveToDatabase(item);

});

}

Nguyên nhân thất bại:

Hàm `forEach` không hỗ trợ các callback không đồng bộ theo cách mà bạn mong muốn. Điều này khiến vòng lặp kết thúc trước khi dữ liệu thực sự được lưu, gây ra sự cố khi kiểm tra kết quả cuối cùng.

Đề xuất sửa đổi:

Sử dụng vòng lặp `for...of` hoặc `Promise.all(items.map(...))` để đảm bảo tất cả các tác vụ không đồng bộ hoàn thành.
